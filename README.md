# 🧠 Gen-AI Lab

This repository contains a series of experiments in **Machine Learning, Deep Learning, and Generative AI**, implemented using Jupyter Notebooks as part of my academic lab work at Maulana Azad National Institute of Technology (MANIT), Bhopal.

---


## 📥 Setup Instructions

1. Clone the repository:

```
git clone https://github.com/singuneet/Gen-AI-Lab.git
```

2. Navigate to the project directory:

```
cd Gen-AI-Lab
```

3. Download the datasets from the links above.

4. Place the datasets in the root directory as shown below:

```
Gen-AI-Lab/
├── DADS dataset/
├── Fashion dataset/
├── Exp_1_WISDM.ipynb
├── Exp_2_water-segmentation-using-vit.ipynb
├── Exp_3_Gaussian Mixture Models.ipynb
├── Exp_4_WISDM_DADS.ipynb
├── Exp_5_LLM.ipynb
├── Exp_6_Fashion MNIST.ipynb
```
